# VUE+LeanCloud 通讯录

> vue通讯录


# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

>对返回数据进行处理，使汉字转拼音用于分组  

>图片转为base64压缩上传  

>城市联动使用了本地json数据  

>简单的做了一个确认框，和提示框插件
